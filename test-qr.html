Тестовые QR-коды для глубоких ссылок

Тест 1: Сессия ID = 8

Отсканируйте этот QR-код камерой приложения или внешним сканером
https://bps-energy.by/qr/start-session/8

Тест 2: Сессия ID = 123

Тест с другим ID сессии
https://bps-energy.by/qr/start-session/123

Тест 3: Сессия ID = 999

Тест с большим ID
https://bps-energy.by/qr/start-session/999

Инструкции по тестированию

  1. Внутреннее сканирование: Откройте приложение → Charging Session → Камера → Отсканируйте QR-код
  2. Внешнее сканирование: Используйте любое приложение для сканирования QR-кодов → Выберите ваше приложение
  3. Прямая ссылка: Скопируйте URL и откройте в браузере → Выберите ваше приложение

Ожидаемый результат: В консоли должно появиться "сессия {id} началась"
