  • Настроить Associated Domains в Apple Developer Portal
  • Разместить файл apple-app-site-association на сервере
  • Установить приложение на устройство

Инструкции по тестированию

  1. Откройте эту страницу в Safari на iOS устройстве
  2. Нажмите на любую из ссылок выше
  3. Должно появиться предложение открыть приложение
  4. Выберите ваше приложение
  5. В консоли должно появиться: "сессия {id} началась"
✅ Ожидаемый результат: При нажатии на ссылку должно появиться предложение открыть приложение, и после выбора приложения в консоли должно появиться сообщение "сессия {id} началась"

Troubleshooting

Если ссылки не работают:
  • Проверьте, что файл apple-app-site-association размещен на сервере
  • Убедитесь, что Associated Domains включена в App ID
  • Переустановите приложение после изменений
  • Проверьте, что Team ID указан правильно
